NXLogRegistry.m

Overview

A singelton for registration of loggers. Every logger created with the static initializers of the NXLogger class will be looked-up or registered with this registry. You don’t really need to use this class directly, unless you create a logger with its non-static initializer and want to reuse it at a later time.
